## Usage

1. Open a Discord call with others screensharing or with a webcam
2. Click the NDI Cord plugin icon
3. Configure stream settings
4. In OBS, add an NDI Source and select your stream
5. Use as a regular OBS source

## Requirements

- Vencord installed and configured
- NDI SDK/tools installed on your system
- OBS Studio (recommended)
- Windows OS (sorry linux, no support just yet)

## Configuration

Configure stream bitrate, resolution, and codec in the plugin settings.
